# Blog Preview Card - Neo-Brutalist Edition

A clean and stylized solution for the classic **Blog Preview Card** challenge from Frontend Mentor. This project showcases a modern article preview component built with clean, semantic web standards.

## The Project

The design follows the *Neo-Brutalist* aesthetic, which features vibrant background colors, crisp thick black borders, and heavy, solid shadows. As a unique personal touch, I implemented a geometric dot-matrix pattern that cascades elegantly into the background corners.

---

## Technologies Used

This project was built from scratch using pure web technologies, focusing on clean code structure and performance:

*   **HTML5:** Structured using strict semantic tags (`<main>`, `<article>`, `<time>`, `<footer>`) to ensure high accessibility for screen readers.
*   **CSS3:** Modern styling utilizing Google Fonts integration, custom layout properties, and smooth micro-interactions.
*   **Flexbox & Grid:** Combined seamlessly to perfectly center the card on the viewport and to handle the mathematical positioning of the background dot-matrix layout.

---

## Key Features

*   **Semantic Accessibility:** Proper use of the `<time>` tag with the `datetime` attribute for optimal screen reader compatibility.
*   **Interactive Hover Effects:** A smooth, animated transition featuring color shifting and a subtle elevation (`translateY`) when hovering over the article title link.
*   **Fully Responsive:** The component scales down smoothly, ensuring an excellent layout on both mobile screens and large desktop viewports.
